In misc.

This is the Third Web Page.

Having now spent a few minutes with this Pelican, I think it is a good idea for publishing some blog, but not right for presenting a collection of documents.

  1. Pages are turned into entries depending on the date tag in them.
  2. Documents can be referred to from the blog entries.
  3. Only the blog entries are converted from Markdown to HTML. (I think Derrick is looking for something that will auto convert Markdown docs to HTML docs. Pelican look like it will convert Markdown docs to blog entries.)
